# RbcParser

## Installing

- The minimum required PHP version is PHP 8.0

```composer
$ composer i
```

```composer
$ php artisan serve
```

## Using routes

```php
GET /parse
GET /news
GET /news/{id}
```

There are migrations and models for the database. A file storage in JSON format is used.
